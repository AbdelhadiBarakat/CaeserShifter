# Caesar Shifter

## Overview
This project is a Python-based encryption and decryption tool that implements the Caesar shifter using a custom 95-character set.

The program allows users to:
- Encrypt messages by shifting characters forward
- Decrypt messages by shifting characters backward
- Use a passcode-based shift system
- Preserve special characters such as symbols and spaces

It demonstrates basic cryptography concepts and modular program design.

## Features
- Encryption and decryption system
- Custom 95-character ASCII-based set
- User input validation
- Passcode-based shift control
- Modular function-based structure
- Preserves non-alphabet characters (symbols, spaces, emojis)

## How to Run
1. Ensure you have [Python](https://www.python.org/downloads/) installed.
2. Clone this repository:
   `git clone https://github.com/AbdelhadiBarakat/CeaserShifter.git`
3. Navigate to the folder:
   `cd CeaserShifter`
4. Run the script:
   `python CeaserShifter.py`
